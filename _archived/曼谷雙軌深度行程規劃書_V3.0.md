# 曼谷雙軌深度行程規劃書 V3.0
**行程期間**：2026年6月25日（週四）— 7月2日（週四）
**住宿基地**：ASAI Bangkok Sathorn（BTS Saint Louis 站）
**行程性質**：印度田野調查後的身心過渡緩衝段
**版本說明**：整合原始規劃 + 網路核實報告 + 壓力測試分析 + Gemini 缺口研究，2026-05-05

---

## 一、雙軌行程架構

```
Phase 1（6/25-6/30）：雙人浪漫文化之旅
  重視氛圍、餐飲體驗、舒適度、高空夜景
  法則：每天只有一個高強度主景，其餘全是氛圍填充

Phase 2（6/30-7/2）：獨旅都市規劃與藝術田野考察
  高強度、深探 2026 曼谷新地標
  聚焦：城市更新、歷史轉型、公共空間設計
```

**全程三項鐵則：**
1. **16:00 硬規則**：沙吞區 16:00 後絕對不叫計程車，只走 BTS/MRT
2. **雨季容錯**：每天主行程備有室內版本（不是「計畫 B」，是「計畫 A 的下雨版」）
3. **身體優先**：從印度返回，身體是最珍貴的資源；任何讓人疲憊的追加項目直接刪除

## 四、Phase 1：雙人浪漫文化之旅（6/25-6/30）

### Day 1（6/25 週四）—— 抵達與身體復原

**主題**：降落、清潔、存在。今天不是觀光日，是印度→曼谷的模式切換日。

**BKK 入境精確動線：**
```
入境大廳（2F Arrivals）
→ 直走，跟著「ARL」黃底黑字指標（拒絕任何 Taxi 仲介搭話）
→ 步行約 350 公尺，下電扶梯至 B1
→ 購票機買「Phaya Thai」單程票（45 THB）
→ 搭 ARL 26 分鐘至 Phaya Thai
→ 出站右轉，穿越 40 公尺露天走廊（雨天先備傘）進 BTS 付費區
→ BTS 往 Bang Wa 方向，4 站至 Saint Louis
```

**若行李多或疲憊至極，改 Grab（推薦）：**
```
入境後 → 2F Exit 4 → 搭電梯至 1F → 出 Gate 4 至室外
→ 找「Grab Booth」→ Pickup Point A 或 B
→ App 定位選「Suvarnabhumi Airport, Level 1, Exit 4, Grab Pickup Point A/B」
→ 不要 pin 在 2F 入境大廳，司機根本過不去
```
給司機的定位說明（英文）：
> `I am at Level 1, Exit 4, Grab Pickup Point A/B, near the Grab booth.`

| 時段 | 行程 | 備註 |
|---|---|---|
| 06:20 | BKK 落地 | TDAC QR Code 截圖備好，海關約 20-30 分鐘 |
| 07:00-08:30 | 前往飯店 | ARL 或 Grab，視行李量決定 |
| 09:00-10:30 | **Health Land 按摩**（步行 5 分鐘） | 預約 90 分鐘泰式按摩（約 500 THB）。比任何景點更值得。 |
| 午後 | 飯店休息 / 庭院泳池 | 等女友抵達，不排景點 |
