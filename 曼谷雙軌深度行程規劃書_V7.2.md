7. **獨旅段保留**：6/29 下午–7/2，研究段（Song Wat + Talat Noi）完整保留。

**主題**：輕量啟動，不急著證明這趟旅行很精彩

```
→ 電梯上 5F 觀景台（免費，看 Lumpini + Sathorn 天際線）
→ BTS 免費接駁（One Bangkok → BTS Phloen Chit，每 15 分鐘一班）
```

**必看：**
- **The Parade B1**：三越超市 + 美食街，MRT 直連，冷氣最強
- **5F 觀景台**：免費，Lumpini Park + Sathorn 天際線全景
- **BTS 免費接駁**：One Bangkok → BTS Phloen Chit，15 分鐘一班

#### 機場交通雙方案

**Option A（推薦）：MRT + ARL**
```
One Bangkok → MRT Lumphini → MRT Si Lom → 走 Silom 地下連通
→ BTS Sala Daeng → Silom Line → Siam → Sukhumvit Line → Phaya Thai
→ ARL 到 BKK
總時長：約 60–75 分鐘
總費用：約 60–80 THB
```

**Option B：Grab 直達**
- 從 One Bangkok 叫 Grab 直接到 BKK
- 約 45–90 分鐘，費用 200–500 THB
- **條件**：下雨/行李重/起飛時間充裕（>3 小時）

#### 導遊解說詞

> One Bangkok 是曼谷新一代「城中城」的代表。它不是單一商場，而是把辦公、零售、酒店、公共藝術、地下連通與車流管理包成一個高端城市介面。最後一天看它很合適，因為它靠 MRT、室內空間、指定上下車點提供高容錯。
>
> 返程日不需要浪漫，不需要冒險，只需要穩。今天你要做的不是再看一個景點，而是安全、準時、情緒穩定地離開曼谷。
>
> 如果早上走了 Green Bridge，把它當成這次獨旅田野的句點——一座新橋把兩個公園縫起來，正好對應你整趟旅行的結構：從雙人段到獨旅段，中間需要一座橋。這趟旅程結束了，但田野筆記才剛開始。

## §8. 交通系統攻略

### 8.1 三層交通思維

| 層級 | 工具 | 適用 |
|---|---|---|
| **主動脈** | BTS Silom/Sukhumvit、MRT Blue、ARL | 跨區移動、避塞車 |
| **毛細管** | Grab、Bolt、計程車 | 最後一哩、行李、下雨 |
| **特殊管道** | Chao Phraya Express Boat、Khlong Boat、飯店接駁 | 河岸、觀光運河、特定活動 |

### 8.2 Rabbit Card 與票卡

- 建議買 **Rabbit Card**（BTS）+ 單程票（MRT）。不建議買一日票除非當天跨區超過 6 趟。
- MRT Blue Line 與 BTS 不共用票卡，換站須重買。
- ARL 單程到 BKK 僅 45 THB，塞車日比 Grab 快一倍。

### 8.3 Grab / Bolt 實務

| 項目 | 建議 |
|---|---|
| 綁卡 | 出發前台灣測試成功一次；V7.0 硬規則 |
| 上車前 | 確認車牌與 App 一致，上車再確認姓名 |
| 費用異常 | 大雨時 Dynamic Pricing 可能翻倍；預估價超過合理範圍就換 Bolt |
| 溝通 | 司機英文程度不穩；關鍵地名用 App 顯示中英對照給司機看 |
| 司機 No-show | 超過預估到達時間 5 分鐘，直接取消重叫，不等 |

### 8.4 塞車地雷時段

| 時段 | 路段 | 對策 |
|---|---|---|
| 週一–五 07:30–09:30 | Silom/Sathorn/Rama IV 進城 | 全走 BTS/MRT |
| 週一–五 **16:30–19:30** | Rama IV / Sukhumvit / Asok | **禁止叫車長程**；走捷運 |
| 週五晚上 18:00–22:00 | 全市 | Grab 預估時間 +50% |
| 雷雨日 | 全市路面積水嚴重 | 強制轉 BTS/MRT |

### 8.5 BKK 機場路線優先序

1. **ARL**（Phaya Thai → BKK）：45 THB，45 分鐘，**最穩**
2. **Grab**：400–600 THB，40–90 分鐘，適合行李重/下雨
3. **計程車排班**：約 400 THB + 高速費 + 50 THB 機場附加，適合深夜

### 8.6 河上交通

| 工具 | 價位 | 用途 |

| Chao Phraya Express Boat（橘旗） | 15–20 THB | 河岸站點移動 |
